# ⛅ 天气套利计算器

Polymarket 天气市场套利机会计算工具

## 功能

- 输入市场 Yes/No 价格
- 输入天气预报概率
- 自动计算优势 (Edge)、期望值 (EV)、ROI
- 给出交易建议

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 部署到 Vercel

1. 推送代码到 GitHub
2. 在 Vercel 导入项目
3. 自动部署完成

## 技术栈

- Next.js 15
- React 19
- Tailwind CSS
- TypeScript
